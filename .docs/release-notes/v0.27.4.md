## Release v0.27.4
*Released at 2026-06-14T09:44:21.595Z*

### Reliability

- **Crash logs are now persisted to disk.** Previously, when the packaged app
  hit an uncaught exception it printed to a stderr that nobody could see (a
  double-clicked executable has no attached console) and then vanished. Crashes
  now append a timestamped stack to `event-horizon-crash.log` next to the
  executable, so failures are diagnosable after the fact.
- **Clear message when the port is already in use.** Launching a second
  instance while another is running (port 3067) no longer crashes with a
  confusing stack — it now reports that the port is in use and exits cleanly.

### Tickets

- **FLUX-492**: Harden engine against silent crashes: persist crash log + handle EADDRINUSE
