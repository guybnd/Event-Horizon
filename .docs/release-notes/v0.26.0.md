## Release v0.26.0
*Released at 2026-06-12T15:17:45.662Z*

### Highlights

- **Agent-facing ticket surface overhaul** — `get_ticket` now returns a digest (session progress collapsed, history windowed, launch args stripped), with a new `get_session_log` escape hatch for raw session detail. Heavily-worked tickets drop from 100k+ chars to a few KB in agent context.
- **Meaningful session logs** — finished sessions store typed milestones and a first-class `finalMessage` instead of hundreds of raw output chunks.
- **Leaner payloads everywhere** — the board list endpoint no longer ships terminal sessions' progress arrays; REST gains `?view=agent` for non-MCP agent reads; oversized ticket bodies get a soft warning.
- **Module system MVP** — first module integrations land, with Serena as the pilot and an expanded BUILTIN_MODULES catalog.
