## Release v0.3.1
*Released 2026-05-11*

This release focuses on performance, reliability, and polish across the board, comment, and token tracking surfaces. Key highlights include smoother comment input, faster ticket modal load times, smarter token cost tracking, and a redesigned new-task experience.

---

### Performance & Responsiveness

- **FLUX-174**: Eliminated choppiness in the comment box for a smooth typing experience
- **FLUX-180**: Fixed blank screen and lag when opening the ticket modal
- **FLUX-188**: Resolved input lag in comment boxes during active agent sessions
- **FLUX-150**: Optimized token spend tracking to reduce unnecessary API overhead

### Token Cost Tracking

- **FLUX-187**: Token cost badge now toggles between cached and fresh token split with color-coded thresholds
- **FLUX-205**: Fixed token counter regression introduced in the last refactor

### Agent Workflow & Grooming

- **FLUX-206**: Fixed grooming skill to properly surface user-input prompts instead of silently skipping them
- **FLUX-194**: Cards now display an animated activity badge while an agent is actively working
- **PROJECT-2**: Reduced per-turn token cost of the agent workflow

### Reliability & Bug Fixes

- **FLUX-189**: Fixed false dirty-state that marked all tickets as modified even when nothing changed
- **FLUX-195**: Fixed tickets silently disappearing from the board after an engine restart
- **FLUX-186**: Fixed code review comments being discarded instead of saved
- **FLUX-181**: Engine now stamps history entry dates server-side for consistent timestamps
- **FLUX-196**: Fixed portal build repeatedly failing due to linter reverting unused-variable fixes
- **FLUX-197**: POST /api/tasks now correctly defaults the project key to the configured project

### Developer Experience

- **FLUX-182**: Split the monolithic engine entry point into focused, maintainable modules
- **FLUX-172**: Code readability improvements and refactor across key surfaces

### UI & UX

- **FLUX-204**: Redesigned new-task creation window with improved UX
- **FLUX-207**: Unread filter pill now shows the count of unread messages
- **FLUX-179**: Internal release infrastructure improvements
