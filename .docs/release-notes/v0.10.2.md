## Release v0.10.2
*Released at 2026-05-29*

### Highlights

Bug fixes and reliability improvements for read-state persistence and real-time notifications.

### Changes

- **Fix read-state write races**: Serialized concurrent writes to the read-state file to prevent data corruption
- **Batch read-state flush**: Portal now batches mark-as-read updates with a 50ms debounce, reducing network requests
- **Mark-all-read SSE broadcast**: Marking all notifications read now broadcasts via SSE so other tabs/clients update instantly
- **Null-notification handling**: Portal gracefully handles mark-all-read events without crashing on null notification payload
- **Storage-sync consolidation**: Stray ticket recovery logic extracted into a dedicated storage-sync module
- **Storage sync fix**: Corrected edge case in storage sync recovery
