## Release v0.11.0
*Released at 2026-05-29T01:49:07.187Z*

### Highlights

**Ticket file integrity hardening** — Atomic file writes prevent corruption from concurrent reads on Windows. Guards in `readTaskFromDisk` and `loadTask` detect partial/corrupt reads and fall back to cache instead of triggering destructive repairs.

**Skill staleness detection** — The engine now compares installed agent skill versions against source on startup and after each agent session. If skills are outdated, a notification with a one-click "Reinstall" button appears in the portal.

**Agent skills v2.3.0** — Overhauled all four skill modules: fixed markdown table rendering, added explicit prohibition against direct file writes to `.flux/` and `.flux-store/`, trimmed unnecessary YAML examples, and added File Boundaries section.

**Textured themes** — New theme presets with texture overlays via CSS tokens.

- **FLUX-341**: Harden ticket file integrity and add skill staleness detection
