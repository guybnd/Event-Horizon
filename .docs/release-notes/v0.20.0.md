## Release v0.20.0
*Released 2026-06-04*

### Highlights

**Multi-agent orchestration** — Launch multiple agents in parallel (scatter-gather), sequential pipeline (relay), or adaptive delegation (supervisor). A unified launcher handles persona selection, template assignment, and phase-aware routing. Single-reviewer shortcut skips the combiner when only one agent is selected.

**Branch-per-ticket workflow** — Tickets can now own a git branch. The portal creates branches on task start, agents check them out automatically, and `finish_ticket` pushes + opens a PR in one step. Branch status (ahead/behind) is visible on cards and in the modal.

**Personas and templates** — Built-in per-phase personas (Groomer, Implementer, Reviewer, Finalizer) and workflow templates ship out of the box. Fork and customize them from the Workflow Builder. Board defaults auto-populate the launcher per phase.

**Documentation overhaul** — Full reference suite: REST API, MCP tools, ticket schema, realtime channels, agent-adapter contract, and task-oriented recipes. Docs restructured by subsystem with a central INDEX.
