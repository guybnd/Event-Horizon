## Release v0.7.0
*Released at 2026-05-19*

### Features

- **Copilot CLI agent integration**: Full rewrite of the Copilot adapter to use the standalone `copilot` CLI binary with JSONL output parsing, cross-platform binary resolution, and proper session management. All three agent frameworks (Claude, Gemini, Copilot) are now fully functional.

### Bug Fixes

- Fix Copilot session start returning 500 (ReferenceError on `exePath`)
- Fix Windows spawn failures with paths containing spaces
- Fix extensionless bash scripts being selected over .exe/.cmd on Windows
- Fix `spawn('node')` resolving to pkg binary instead of system Node.js

### Technical Details

- Cross-platform Copilot binary resolution: PATH → VS Code globalStorage → npm global
- Bypass `.cmd` wrapper chain on Windows by spawning `node npm-loader.js` directly
- JSONL event parsing for `assistant.message_delta`, `assistant.tool_call_start`, `assistant.reasoning_delta`, and session lifecycle events
- Added `copilotCli` integration config (grooming/implementation model selection)
