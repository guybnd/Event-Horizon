## Release v1.6.1 — first-run reliability on macOS, sync self-healing, and a hardened release pipeline
*Released at 2026-07-13T16:10:11.701Z*

### Highlights

A reliability-focused patch driven by real first-install reports from the packaged macOS app, plus a sync escape hatch and release-pipeline hardening.

- **Packaged macOS app: first-run actually works** — a GUI-launched (dmg) app inherits launchd's minimal PATH, which hid Homebrew git, `gh`, the `claude` CLI, and `serena` from the engine. The engine now resolves the user's login-shell PATH once at startup (FLUX-1408). Git Sync onboarding no longer requires git ≥ 2.42: orphan-branch creation falls back to version-agnostic plumbing, retries converge instead of erroring, and failures name the exact git binary/version the engine ran (FLUX-297, FLUX-1410). The "Install the integration" step surfaces its real error instead of a generic banner, and packaged-mode errors are mirrored to a findable log file (FLUX-1407). Windows: the portal terminal now spawns ComSpec/PowerShell instead of falling back to `/bin/bash` (FLUX-1404).
- **Sync self-healing** — a wedged `flux-data` sync (diverged, half-merged, or corrupted store) now has a force-reset-to-remote escape hatch, as both a CLI and a one-click portal action, instead of requiring manual git surgery (FLUX-1232).
- **Furnace & agent-session reliability** — an expired API/OAuth token is now classified as transient: the batch halts with one "re-auth needed" notification instead of N opaque per-ticket hard-fails, and tool-originated HTTP 403s no longer false-trigger that path (FLUX-1397, FLUX-1406). Dirty worktrees on terminal tickets are detached instead of silently hogging Furnace slots forever (FLUX-1405).
- **Release pipeline hardening** — version tags are immutable on the public remote (no more two-builds-under-one-tag), publishing refuses a dirty/diverged worktree, and CI keeps a release draft until the core artifact set is verified attached (FLUX-835). Both architecture ratchets are green again: the v1.6.0 adapter-boundary leak is fixed (FLUX-1412) and stray bare git spawns were routed through the hardened runner (FLUX-1403).
- **Cost observability** — the cost gauge is accurate again: estimator model-hint bug, cache-read pricing, resume roll-up, and a badge breakdown (FLUX-1375).
- **Design & planning craft** — a conventional design style-guide doc with a bootstrap-from-code flow (FLUX-1399), and plan artifacts now get design guidance injected at all three publish hooks (FLUX-1398).
