## Release v0.21.0
*Released 2026-06-05*

### Highlights

**Relay and supervisor orchestration modes** — Two new multi-agent topologies: relay pipelines pass context through a chain of agents sequentially (ideal for TDD or staged refinement), and supervisor mode delegates work adaptively through MCP tool calls. Both integrate with the existing launcher UI.

**Prompt-prefix caching for reviews** — Scoped git diffs are injected into a cacheable prompt prefix so scatter-gather review sessions reuse context across workers, cutting token spend significantly.

**Dev-runner resilience** — Graceful restart defers engine reload while agent sessions are active. A predev shutdown hook kills the previous engine instance to prevent port conflicts.

**Onboarding and installer fixes** — Storage migration no longer errors when `flux-data` branch already exists locally. Skill reinstall from notifications now resolves the correct source directory.
